# vuln-checker
Report vulnarability checker

# How to use? 

## 1. npm install
## 2. put all archived (.zip) reports into 'reports' folder
## 3. npm run start

# Result
## After the check you will see content in 2 folders
## - 'empty' folder 
## - 'has-vulns' folder
